Quickly inject a random admin user for Moodle 5.0+

Not intended for malicious use!

Upload inject_moodle_admin.php to your site`s public_html folder then navigate to for example https://yourdomain.com/inject_moodle_admin.php 

The script will inject and print a random Admin user then delete itself - doulbe check the public_html folder once user is generated to ensure the script is deleted.
